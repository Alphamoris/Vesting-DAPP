## 🚀 **Getting Started**

### 📋 **Prerequisites**

Before you jump into Oggy's world, make sure you have:

```bash
# Required tools
Node.js 18+ 📦
Rust 1.70+ 🦀
Solana CLI 1.16+ ⚡
Anchor Framework 0.31+ ⚓
```

### ⚡ **Quick Start**

```bash
# 1. Clone the repository
git clone https://github.com/Alphamoris/Vesting-DAPP.git
cd Vesting-DAPP

# 2. Install dependencies
npm install

# 3. Build the Anchor program
npm run anchor-build

# 4. Start local validator
npm run anchor-localnet

# 5. Deploy to local network
npm run anchor-deploy

# 6. Start the development server
npm run dev

# 🎉 Open http://localhost:3000 and start exploring!
```

### 🔧 **Development Commands**

```bash
# Frontend Development
npm run dev          # Start development server
npm run build        # Build for production
npm run lint         # Check code quality

# Smart Contract Development
npm run anchor-build    # Compile smart contracts
npm run anchor-test     # Run contract tests
npm run anchor-deploy   # Deploy to network

# Code Quality
npm run format         # Format code with Prettier
npm run ci            # Run complete CI pipeline
```

## 🚀 **Deployment**

### 🌐 **Mainnet Deployment**

```bash
# 1. Configure for mainnet
anchor keys sync
anchor build

# 2. Deploy to mainnet
anchor deploy --provider.cluster mainnet-beta

# 3. Update frontend configuration
# Update NEXT_PUBLIC_RPC_ENDPOINT in .env.local

# 4. Deploy frontend
vercel --prod
```

### 🧪 **Devnet Deployment**

```bash
# Deploy to Solana devnet
anchor deploy --provider.cluster devnet
```
